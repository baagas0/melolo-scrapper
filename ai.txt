Generate access token with password method
post https://api.dailymotion.com/oauth/token
grant_type
string
required
Defaults to password

set to password to specify the grant type flow
client_id
string
required

The API key retrieved from your Dailymotion Studio.
client_secret
string
required

The API secret retrieved from your Dailymotion Studio.
username
string
required

the username of the Dailymotion channel you want to access
password
string
required

the password of the Dailymotion channel you want to access
scope
string
required
Defaults to manage_videos

needs to be defined for specific permissions or access rights (manage_videos)
